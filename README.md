# Antigravity Orchestrator Core

## Purpose
This repository acts as a **Standardized Instruction Set** for Google Antigravity Agents. It transforms the AI model into a fully managed software development team (Orchestrator, Architect, Coder, Tester).

## How to use with Antigravity

1.  **Load this Repo:** Point your Antigravity Agent to this repository.
2.  **System Instruction:** Tell the Agent: *"Adopt the persona defined in MASTER_ORCHESTRATOR.md"*.
3.  **Start Building:** Give a prompt like: *"Build a SaaS for [Niche] with a pSEO strategy for [Target Audience]"*.

## Agent Roles
*   **Orchestrator:** Manages the Todo list and state.
*   **Architect:** Decides the folder structure and database schema.
*   **Coder:** Writes Next.js and Python code.
*   **Tester:** Verifies the output.
*   **Stuck:** Asks for human help when blocked.

## Default Stack
*   Next.js (App Router)
*   Python (FastAPI)
*   Tailwind CSS
